数据库: Redis+Mongodb, Mysql集群

数据源：Tushare

# 期货回测+交易系统

* 支持多合约回测，多合约组合成为一个Portfolio进行独立回测
* 支持多策略回测，在一个Account下面可以组合多个Portfolio同时回测
* 支持Ticker级别的回测
* 项目已经实现回测和交易系统的统一，可以用同一套策略代码实现回测和交易，目前交易系统尚未开源，有需要的可以加入技术交流群联系作者

**开发环境**: `MacOS`, `Python3.5 Python3.6`
